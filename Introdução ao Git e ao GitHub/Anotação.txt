O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos básicos do Git:

Git Init: Server para começar um projeto que ainda não seja um repositório.

Git clone: É um comando para baixar o código-fonte existente de um repositório remoto.

Git status: Comando utilizado para verificar o status de projetos no Git Bash.

Git commit: Serve para finalizar as alterações realizadas e criar um hash.

Git push: Transcreve todas alterações realizadas para o repositório dentro do GitHub.
